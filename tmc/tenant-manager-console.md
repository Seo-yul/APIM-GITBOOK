# 사용자 관리콘솔이란?

APIM의 사용자 관리 콘솔 (Tenant Manager Console)에서는 APIM에서 사용하는 모든 사용자 계정과 프로젝트를 관리할 수 있습니다.

사용자 관리 콘솔에서는 시스템 사용자 관리와 프로젝트 관리 두 가지 메뉴로 나누어 기능을 확인할 수 있습니다.
